Pokébot

Welcome

Pokébot is a bot for Pokémon battles in discord, built using the Discord4J framework. It has only the core gameplay, as in YOU are the Pokémon. There are none to catch and evolve, and no Pokémon is better than another. You are free to create whatever combinations you want! Using either commands or the responsive web interface, it's easy to set any stats, moves, abilities, level, and natures you want!

How can I use it?

There's an official bot up, which can be added from here. You can find information about how to use it here.

Preventing OP

To help prevent someone from choosing only the "best" moves and maxing their stats, there's this concept of Points. You can only have so many points, and going over them prevents you from doing the fun stuff. Feedback is appreciated by telling me either here (@Coolway99) or on discord (Coolway#7694).

What's planned

It may seem small at first, but Pokébot already has a lot to offer. More Moves and Abilities are being added all the time, with Items, team battles, "slots" and "slot" battles (which are more like the traditional battles), and much more! Leaving feedback for an idea is a sure way to get it added.
